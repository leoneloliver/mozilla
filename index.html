 Web Performance Hackathon 2018  

Mozilla July 14, 2018

The Web Perfornce Hackathon is meant to give a clear view about Firefox OS platform and encourage people to develop Progressive Web Apps.

(PWA)Progressive web apps use modern web APIs along with traditional progressive enhancement strategy to create cross-platform web applications. These apps work everywhere and provide several features that give them the same user experience advantages as native apps. This set of docs tells you all you need to know about them.

Who’s invited
The Web Performance Hackathon is a closed event for members of Mozilla Web Performance at the Toronto Mozilla office. @mocotoOH @mozilla.

The Chanllenge
Developers hacking around web pages performance as part of Mozilla's mission for suporting Progressive Web Apps. Participants were challenged to develop a PWA for Firefox Platform following web performance best practices.
